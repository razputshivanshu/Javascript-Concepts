JavaScript Data Types

JavaScript has two main categories of data types:

  • Primitive Data Types
  • Non-Primitive (Reference) Data Types

Primitive Data Types

Type Description Example
Number Represents numerical values, both integers and floating-point numbers. let age = 25;
String Represents a sequence of characters. let name = "John";
Boolean Represents logical values: true or false. let isStudent = true;
Undefined A variable that has been declared but not assigned a value. let x;
Null Represents the intentional absence of any value. let y = null;
Symbol Represents a unique and immutable value, often used as object keys. let sym = Symbol("unique");
BigInt Used to represent numbers larger than Number.MAX_SAFE_INTEGER. let big = 12345678901234567890n;

Non-Primitive Data Types

These data types are also called reference types and include:

Type Description Example
Object Represents a collection of key-value pairs. let person = { name: "Alice", age: 30 };
Array A special type of object used to store ordered collections. let numbers = [1, 2, 3];
Function A block of reusable code. function greet() { return "Hello"; }
